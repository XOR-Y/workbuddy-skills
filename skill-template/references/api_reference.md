# API 参考文档模板

此文件是参考文档模板，用于存放需要在 Skill 中引用的详细文档。

## 使用说明

1. 将相关的 API 文档、规范或参考资料放在此文件中
2. 在 SKILL.md 中引用此文件：`详细规范参见 \`references/api_reference.md\``
3. 如果文档很大（>10k 词），考虑拆分成多个文件

## 示例内容结构

### API 端点

#### 端点 1: `/api/resource`

**方法**: GET

**参数**:
- `param1` (string, 必需): 参数描述
- `param2` (number, 可选): 参数描述

**响应**:
```json
{
  "data": "..."
}
```

### 数据模型

#### 模型 1

| 字段 | 类型 | 描述 |
|------|------|------|
| id | string | 唯一标识符 |
| name | string | 名称 |

### 错误处理

| 错误码 | 描述 |
|--------|------|
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 404 | 资源不存在 |
